# OpenFork TODOs

## Root path (`/`) — Public workspace discovery

Currently `/` redirects to `/new`. In the future, this should become a workspace
discovery page supporting:

- **Public workspaces** — visible to everyone, can be joined freely
- **Semi-private workspaces** — visible but require a request to join
- **Workspace search** — find workspaces by name or topic

This would give unauthenticated users a landing experience and let existing
users discover and switch between workspaces.
